What is the definition of server port?

This is a protocol that runs on many computers and is used to authenticate users of TCP connections. 
Information about a large number of computers can be obtained using standard such services. 
But it can be used as a logger for many services, especially FTP, POP, IMAP, SMTP, and IRC.

• What is server port range?

0 to 65535

• How many bits does number above represent?

5 Bytes
